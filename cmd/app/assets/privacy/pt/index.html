Política de Privacidade - TopStrem 1. Introdução Esta política descreve como o TopStrem trata as informações dos usuários. 2. Coleta de Informações Não coletamos dados pessoais sensíveis (nome, endereço, telefone, documentos). Coletamos apenas dados técnicos anônimos: estatísticas de uso, modelo do dispositivo, sistema operacional, endereço IP, tipo de rede. 3. Uso das Informações Usamos os dados exclusivamente para melhorar a experiência, garantir o funcionamento, monitorar desempenho e corrigir falhas. 4. Compartilhamento Não compartilhamos, vendemos ou cedemos dados pessoais a terceiros. Podemos usar serviços analíticos (Google Analytics, Firebase) conforme suas políticas. 5. Conteúdo Exibido O TopStrem exibe capas e informações de filmes/séries de APIs públicas. Não hospedamos nem distribuímos conteúdo protegido por direitos autorais. 6. Segurança Adotamos medidas técnicas para proteger os dados coletados. 7. Direitos do Usuário Você pode solicitar informações ou exclusão dos seus dados. 8. Contato E-mail: suporte@topstrem.com 9. Alterações Esta política pode ser atualizada. Consulte esta página regularmente. Última atualização: Abril de 2026
